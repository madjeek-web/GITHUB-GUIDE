## Français

### Versions supportées

| Version | Support |
|---|---|
| Dernière version sur `main` | Oui |
| Versions archivées | Non |

### Signaler une vulnérabilité

Si vous découvrez une faille de sécurité dans ce projet, merci de ne pas l'exposer publiquement via une issue GitHub.

**Procédure à suivre :**

1. Envoyez un e-mail à l'adresse disponible sur le profil GitHub : [https://github.com/madjeek-web](https://github.com/madjeek-web)
2. Décrivez la vulnérabilité de façon précise : nature du problème, étapes pour le reproduire, impact potentiel.
3. Si possible, proposez une correction ou une piste de solution.

**Engagement de réponse :**

- Accusé de réception dans les 48 heures.
- Evaluation du rapport sous 7 jours.
- Communication sur l'avancement et la correction prévue.

### Portée

Ce dépôt contient principalement de la documentation et des exemples de code pédagogique. Les risques de sécurité critiques sont limités, mais tout signalement est pris au sérieux.

## English

### Supported versions

| Version | Support |
|---|---|
| Latest version on `main` | Yes |
| Archived versions | No |

### Reporting a vulnerability

If you discover a security flaw in this project, please do not expose it publicly via a GitHub issue.

**Steps to follow :**

1. Send an email to the address available on the GitHub profile : [https://github.com/madjeek-web](https://github.com/madjeek-web)
2. Describe the vulnerability precisely : nature of the problem, steps to reproduce it, potential impact.
3. If possible, propose a fix or a solution path.

**Response commitment :**

- Acknowledgement within 48 hours.
- Evaluation of the report within 7 days.
- Communication on progress and planned correction.

### Scope

This repository contains mainly documentation and educational code examples. Critical security risks are limited, but all reports are taken seriously.
